> 本实现基于任务目标先构建可拓展原型，并预留内容替换位。

## 子 Agent 组织（公司内部）

1. **NarrativeAgent（叙事）**
   - 维护章节、事件文案、对话节奏
2. **CombatAgent（战斗）**
   - 维护技能、敌人、数值平衡
3. **FrontendAgent（表现）**
   - 维护 UI、动画、布局
4. **AssetAgent（美术资源）**
   - 管理 AI 图像生成与透明背景规范
5. **QAAgent（质量）**
   - 维护基本回归检查与可玩性

## 产品原则

- 可拓展优先：数据与逻辑分离
- 模块边界清晰：系统解耦、事件驱动
- 美术可替换：统一资源路径与角色ID映射
